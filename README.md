Microbial lung-to-blood translocation associates with systemic inflammation in severe pneumonia
This repository contains all code necessary to reproduce the main analyses, main figures, tables, results, as well as all supplemental figures and tables from the manuscript:
"Microbial lung-to-blood translocation associates with systemic inflammation in severe pneumonia: Evidence from paired plasma and lower respiratory tract metagenomics"
Yang et al., 2025

Overview
Purpose: The scripts in this repository will reproduce all primary and supplemental statistical analyses and visualizations corresponding to the published manuscript, including figures and tables.

Content: Code covers complete workflows for data preparation, statistical tests, and figure/table generation described in the manuscript.

Data: Curated data required for these analyses will be provided by the investigators upon reasonable request. The underlying raw data are available in public repositories as described in the Data Availability section of the published manuscript.

Instructions
Clone or download this repository.

Request curated data access from the study investigators if not already obtained.

Provide raw and/or curated data as indicated in the code headers and manuscript Methods section.

Run scripts in R as specified to reproduce figures, tables, and results.

Citation
When using or adapting this code, please cite:
Yang H, Hensley MK, Nguyen V-DB, et al. Microbial lung-to-blood translocation associates with systemic inflammation in severe pneumonia: Evidence from paired plasma and lower respiratory tract metagenomics. (2025).
